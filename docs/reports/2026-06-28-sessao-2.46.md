## Decisões técnicas

- `escalate_to_human` não passa por `assertToolAllowed()` — é meta-controle, sempre disponível
- `AnthropicLlmAdapter` filtra `ThinkingBlock` antes de mapear content (SDK retorna tipos extras)
- `PendingActionService` sem opt-in: é infra de dados local, zero dependência externa
- Retomada é fire-and-forget: erro no callback não falha o `approve()`
- `PendingActionPrismaClient` interface mínima: testabilidade sem acoplar ao `PrismaClient` real
