## O que foi feito

- Redesenhado runner stage: instala prod deps + roda `prisma generate` in-place; não copia node_modules do builder
- Build final: `✔ Image docker-governa-core Built` (122s)
- `docker compose up -d` → todos os 3 containers `Healthy`/`Started`
- `docker logs governa_core`: `No pending migrations to apply` + `servidor ouvindo na porta 3000` ✅

## Decisões tomadas
- **D1** — Runner roda `prisma generate` in-place em vez de copiar do builder: mais simples, determinístico e não depende do layout interno do pnpm store
- **D2** — `prisma` movido para `dependencies`: é necessário em runtime (`migrate deploy`) e não pertence em devDeps
- **D3** — `openssl` instalado via `apk` apenas no runner: builder não precisa (apenas compila TS); runner executa `migrate deploy` que carrega o schema engine binário
