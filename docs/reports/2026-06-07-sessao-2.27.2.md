## TL;DR

Sessão intermediária para diagnosticar e corrigir os erros do CI governa-ui. Identificado `index.lock` bloqueando git (removido pelo usuário), verificado CI #17 vermelho com causa raiz `Cannot find module '@angular/common/locales/pt-BR'` nos specs de clientes-list e cliente-card. Fix aplicado em `jest.config.js`: entrada `moduleNameMapper` para `@angular/common/locales/*` que bypassa a resolução de subpath exports do pnpm. Commit `c5ee4d0` pushed — aguarda CI #18 para confirmar verde.

## Estado

**Parcial** — fix aplicado e pushed (`c5ee4d0`), aguarda CI #18.

## O que foi feito

- Boot protocol completo: leitura do report 2.27.1
- Diagnóstico do `index.lock` em `.git/`: criado pelos comandos `git diff origin/main..HEAD` e `git remote show origin` (sandbox sem SSH key → operação interrompida → lock não limpo)
- Remoção manual dos locks pelo usuário: `index.lock` e `HEAD.lock`
- Code review estático de todos os 8 spec files do governa-ui (todos ok estruturalmente)
- Análise do log real do CI #17: causa raiz identificada — `Cannot find module '@angular/common/locales/pt-BR'` em `clientes-list.component.spec.ts` e `cliente-card.component.spec.ts`
- Root cause: pnpm workspace não hoista subpaths de `@angular/common/locales/*` de forma acessível pelo resolver padrão do Jest (exports field do package.json não é seguido corretamente via symlinks pnpm)
- Fix aplicado em `apps/governa-ui/jest.config.js`: adicionada entrada `moduleNameMapper`:
  ```
  '^@angular/common/locales/(.*)$': '<rootDir>/node_modules/@angular/common/locales/$1'
  ```
- Commit `c5ee4d0` pushed pelo usuário via terminal

## Decisões tomadas

- **D1** — Não tentar rodar testes via sandbox: node_modules macOS têm esbuild incompatível com Linux (sandbox). CI é a única forma válida de validação.
- **D2** — Fix via `moduleNameMapper` e não via mudança nos spec files: mais cirúrgico, não altera lógica de teste nem assertions.
- **D3** — Commits feitos via terminal do usuário (sandbox bloqueia push SSH e gera HEAD.lock/index.lock não-removíveis).
