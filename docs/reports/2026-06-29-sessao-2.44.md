- **`toolsets=all`** no endpoint MCP — habilita todas as tools disponíveis (logs, APM, metrics, infra, synthetics). Pode ser restringido depois com `?toolsets=logs,apm` se necessário

- **API Key + Application Key** no header em vez de OAuth — OAuth exigiria nova sessão para completar o browser flow; header auth funciona imediatamente e é suportado pelo Datadog MCP
- **`DD_API_KEY` primeira fornecida era UUID inválido** — usuário confundiu com Organization ID; corrigido com chave de 32 hex chars
