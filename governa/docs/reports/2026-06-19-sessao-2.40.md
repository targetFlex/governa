## Problema estrutural identificado

O agente não tem acesso aos logs do GitHub Actions. O MCP `github-official` expõe apenas: arquivos, PRs, issues, commit statuses. **Não expõe** workflow runs, logs de CI, checks.

Isso obriga o usuário a colar logs manualmente — fonte de improdutividade e atrasos.
